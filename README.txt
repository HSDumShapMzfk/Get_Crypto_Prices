Простое оконное приложение, написанное на языке python, с использованием фреймворка PyQt6, и бесплатного API - CoinGecko API. 

В приложении есть/планируется добавить следующий функционал:
	- отслеживание волатильности списка криптовалют в реальном времени
	- возможность добавить пользовательскую пару валюта-криптовалюта
	- данные о рыночной капитализации, объем торгов, процент изменения за 24 часа и др.

Для запуска приложения: откройте командную строку -> перейдите в директорию проекта -> введите команду "pip install -r requirements.txt" - установка зависимостей -> введите команду "python main.py" - запуск приложения.

Это мой первый опыт работы с фреймворком PyQt6. Проект не претендует на звание самого чистого/оптимизированного/удобного/гибкого приложения.
